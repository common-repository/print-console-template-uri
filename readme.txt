== Description ==


Prints in the console which template is used if user is logged in.

== Installation ==


This section describes how to install the plugin:


1. Upload the `print-console-template-uri` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress



== Usage ==

Activate the plugin and when logged in. View console when you need to know current page's template uri.
